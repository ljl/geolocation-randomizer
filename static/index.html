geolocation-randomizer

Generates random geolocations for you

Usage

Get random locations around an area

Method: POST
Path: /api/area
Body:
{
  "lat": 59.2,
  "long": 10.1,
  "radius": 10,
  "count": 10
}
Method: GET
Path: /area/api/59.2/10.1/10/10

Get a single random location

Method: GET
Path: /api/random

Get 100 random locations

Method: GET
Path: /api/random/100
